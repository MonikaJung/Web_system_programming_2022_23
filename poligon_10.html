Rodzaje mediów:

  • all - dla wszystkich typów urządzeń
  • aural - dla syntezatorów mowy i dźwięku
  • braille - dla urządzeń przeznaczonych dla niewidomych, które umożliwiają czytanie
  • embossed - dla drukarek brailla
  • handheld - bezprzewodowe urządzenia ręczne
  • print - dla podglądu wydruku oraz materiałów, które zostaną wydrukowane
  • projectio - dla prezentacji projektorowych
  • screen - dla komputerów z kolorowym ekranem
  • tty - dla dalekopisów, terminali albo przenośnych urządzeń z ograniczonymi możliwościami wyświetlania
  • tv - dla telewizorów

Reguła@media

Pozwala definiowć style nie w zależnośći od konkretnych elementów strony lub ich zawartości, ale biorąc pod uwagę typ urządzenia, na którym jest wyświtlana.

Przykłady użycia dla tła:

Ten element div dla mediów typu all powinien mieć błęitne tło.
Ten element div dla mediów typu screen powinien mieć czerwone tło.
Ten element div dla mediów typu tv powinien mieć fioletowe tło.
Takie jak tutaj :)
Ten element dla mediów typu all też powinien być błękitny - chyba że są one równocześnie mediami typu screen. Wtedy tło będzie czarne, a tekst biały. Warto zauważyć, że styl czcionki italic był przypisany do stylu dla mediów typu all, a nie screen, zatem widać, że pierwszy styl faktycznie został zastosowany, a tylko kolidujące ze sobą elementy zostały podmienione na te ze stylu dla mediów typu screen.

Z każdy innym aspektem nakładania styli będzie podobnie - ustawianie wielkośći tabel, obrazów, rodzajów tekstów, wielkośći czcionek, kolorów, itd. itd.
